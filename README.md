🌱 Plant-O8 – Smart Plant Care & Monitoring System








📌 Overview
Plant-O8 is a web-based platform designed to simplify plant care for everyone. It helps users maintain healthy plants by providing personalized care guides and a basic monitoring system. The goal is to make plant care easy, efficient, and accessible, even for beginners.







❗ Problem Statement
In today’s busy lifestyle, people often keep plants but struggle to care for them properly due to lack of knowledge and time. Incorrect watering, sunlight exposure, and delayed detection of issues result in poor plant health.






💡 Solution
Plant-O8 allows users to enter their plant type and instantly receive a customized care guide including watering schedule, sunlight requirements, and maintenance tips. It also provides a monitoring feature to help users track plant health over time.







🚀 Features
🌿 Personalized Plant Care Guide
📊 Plant Monitoring System
🖥️ Simple & User-Friendly Interface
🗄️ Data Storage using MongoDB
